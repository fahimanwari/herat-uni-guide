# بخش ۱: نتیجه تحقیق انترنتی — سوالات واقعی کجاست؟ (۱۸ جولای ۲۰۲۶)

## ۱.۱ فورمت رسمی امتحان (تایید شده)

- کانکور عمومی: **۱۶۰ سوال چهارجوابه**، **نمره کل ۳۶۰**، حدود **۳ ساعت** — برگزارکننده: اداره ملی امتحانات (NEXA) — [nexa.gov.af](https://nexa.gov.af/fa/index)
- ⚠️ **تعداد دقیق سوال هر مضمون به‌صورت رسمی منتشر نمی‌شود** و بین فورم‌ها کمی فرق دارد. تنها راه دقیق: شمارش از روی PDF یک فورم واقعی (منابع پایین). blueprint فعلی سیستم با تخمین متعارف seed شده و باید در فاز ۲ با شمارش واقعی اصلاح شود.

## ۱.۳ دو واقعیت مهم که پلان باید با آن روبه‌رو شود (صادقانه)

1. **فورم‌های کامل ۱۴۰۲–۱۴۰۵ به‌صورت رسمی آنلاین منتشر نشده‌اند.** NEXA فورم‌ها را عمومی نمی‌کند. راه واقعی: کتاب‌های بازار + کانال‌های تلگرامی + همکاری با مراکز آمادگی کانکور هرات. **هیچ اسکریپت یا AI جای تایپ و بازبینی انسانی این بخش را نمی‌گیرد** — این کار محتوایی است، نه فنی.
2. **کات‌آف ۱۴۰۵:** برچسب سیستم دینامیک است — الان آخرین سال موجود در DB را نشان می‌دهد (فعلاً ۱۴۰۴ از seed). وقتی نتایج ۱۴۰۵ منتشر شد، ادمین فقط کات‌آف‌ها را از پنل وارد می‌کند و برچسب خودکار «سال ۱۴۰۵» می‌شود. **هیچ تغییر کدی لازم نیست.**

## ۱.۲ قوانین ورود دیتا (برای هر سوال — بدون استثنا)

```json
{
  "subject": "ریاضی",              ← نام مضمون دقیقاً یکدست: ریاضی، فزیک، کیمیا، بیولوژی، دری، پشتو، تعلیمات اسلامی، تاریخ، جغرافیه، انگلیسی
  "year": 1398,                    ← سال فورم واقعی (الزامی برای سوالات کانکوری)
  "grade": "11",                   ← صنف کتاب درسی منبع سوال: "10" یا "11" یا "12" (سیستم فقط همین ۳ مقدار را قبول می‌کند)
  "chapter": "مثلثات",             ← فصل/باب کتاب (اختیاری ولی بسیار مفید برای آزمون موضوعی آینده)
  "source": "کانکور ۱۳۹۸ — آرشیف انیس",  ← منبع دقیق؛ اگر از کتاب بود نام کتاب و صفحه
  "difficulty": "medium",
  "question_fa": "...",
  "options": [ {"text":"...","is_correct":true}, {"text":"..."}, {"text":"..."}, {"text":"..."} ],
  "explanation_fa": "..."          ← حل تشریحی — برای صفحه مرور بعد از آزمون
}
```

- سیستم خودش رد می‌کند: کمتر از ۲ گزینه، بدون جواب درست، بیشتر از یک جواب درست، صنف غیر از ۱۰/۱۱/۱۲. ورود «همه یا هیچ» است — اگر ردیف ۵۰ خطا داشته باشد هیچ‌چیز ثبت نمی‌شود و شماره ردیف خطا نشان داده می‌شود.
- [ ] **کنترل کیفیت دومرحله‌ای (قانون طلایی):** واردکننده ≠ بازبین. سوال با `is_verified: false` وارد شود؛ نفر دوم با PDF/کتاب اصلی مقایسه کند و تیک «تایید شده» را در پنل بزند. **فقط سوالات تایید شده وارد آزمون می‌شوند — این در کد enforce شده است.**
- [ ] تقسیم کار: هر نفر یک «مضمون-سال»؛ هدف ۲۰۰ سوال/هفته/نفر
- [ ] پیشرفت را از داشبورد پنل → آمار بانک (به تفکیک سال و صنف) دنبال کنید — همین حالا کار می‌کند

### ۴.۱.د — اتصال به هوش مصنوعی (تا مشاور واقعاً «رهنمای جامع» شود)

- [ ] در `ai/indexer.py` متد `_department_to_text` گسترش یابد: `intro_fa` + کریکولم سمستروار («مضامین سمستر اول: ...») + عنوان ویدیوها و نام استادان به متن ایندکس اضافه شود → بعد reindex، چت‌بات به سوالاتی مثل «در سمستر اول کمپیوتر ساینس چه می‌خوانم؟» یا «آیا انجنیری برای من مناسب است؟» با دیتای واقعی جواب می‌دهد

### ۴.۱.هـ — جریان جمع‌آوری ویدیوهای ترفیع استادان (کار سازمانی — موازی با کدنویسی شروع شود!)

ایده: هر استادی که ترفیع علمی می‌خواهد باید ویدیوی درسی یک مضمون بسازد — این ویدیوها همین حالا تولید می‌شوند و بهترین محتوای رایگان برای ماست.
- [ ] ۱. نامه رسمی به معاونیت علمی پوهنتون: درخواست دسترسی به آرشیف ویدیوهای ترفیع + **اجازه کتبی نشر عمومی** (حق نشر مال استاد/پوهنتون است — بدون اجازه کتبی هیچ ویدیویی منتشر نشود)
- [ ] ۲. از شبکه ۱۶ رابط پوهنځی (پوشه `question/`) هر رابط: لیست ویدیوهای موجود پوهنځی خودش + رضایت استاد
- [ ] ۳. معیار انتخاب: کیفیت صدا قابل فهم + مضمون از سمسترهای ۱-۲ (برای شاگرد مکتب قابل هضم) + ترجیحاً زیر ۳۰ دقیقه
- [ ] ۴. آپلود: کانال YouTube رسمی پروژه (unlisted کافی است) → لینک + نام استاد + مضمون + سمستر در پنل ثبت شود — نام استاد همیشه ذکر شود (انگیزه مشارکت!)
- [ ] ۵. هدف موج اول: **هر پوهنځی حداقل ۱ ویدیو** (۱۶ ویدیو) → موج دوم: هر دیپارتمنت فارغ‌ده ۱ ویدیو
- [ ] ۶. کریکولم سمستروار: از همان رابط‌ها — عکس/فایل کریکولم رسمی دیپارتمنت را بگیرند؛ تیم محتوا به فورمت JSON بالا وارد کند

**معیار پذیرش ۴.۱:** برای ۳ دیپارتمنت نمونه (مثلاً کمپیوتر ساینس، طب، حقوق): صفحه رشته intro + کریکولم ۴ سمستر + حداقل ۱ ویدیوی استاد را نشان دهد · چت‌بات به «سمستر اول X چه می‌خوانم؟» درست جواب دهد · شاگرد از صفحه رشته با یک کلیک به آزمون علاقه برسد · هیچ ویدیویی بدون اجازه کتبی منتشر نشده باشد.
